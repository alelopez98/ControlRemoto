Remote Desktop Protocol(RDP)

  • Aplicación exclusiva de Windows.
  • Permite la ejecucion de una aplicacion entre un terminal y un servidor.
  • Información comprimida.
  • Cifrado de 128 bits, algoritmo RC4 y TLC.

Practica Nº1 (PC -> PC)

  • 1.- Descargar TeamViewer.
  • 2.- IP: 192.168.2.176 usuario: asir contraseña: asir.
  • 2.- Intalación (opcional): uso personal.
  • 3.- Introducir ID y contraseña del equipo al que se quiere acceder.

Practica Nº2 (Movil -> PC)

  • 1.- Acceder a Play Store.
  • 2.- Instalar TeamViewer.
  • 3.- Introducir ID y contraseña del equipo a conectar.
  • 4.- (Opcional) Descargar TeamViewer Quicksupport (PC -> Movil).
  • 5.- Descargar versión de nuestro SO.
  • 6.- Introducir en el equipo la ID del movil.
